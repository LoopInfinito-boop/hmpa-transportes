## Funcionalidades

- Login com perfis: Solicitante, Motorista e Gestor
- Formulário de demanda em 4 steps (Tipo → Rota → Dados → Confirmar)
- Dados completos do paciente para ambulância
- Lista de itens para carro de apoio
- Painel do motorista com fila priorizada (Emergência → Urgente → Normal)
- Registro de KM inicial e KM final com cálculo automático
- Histórico completo de corridas
- Relatório financeiro por veículo
- Notificações entre usuários
- Cadastro de setores e veículos

## Tecnologia

HTML5 + CSS3 + JavaScript puro. Dados salvos no `localStorage` do navegador.
